# draggable-picture
You can drag a picture in an element,like div.

## 1 介绍

这是一个使用原生javascript实现类似淘宝商品详情页放大镜效果的简易demo，两种显示模式：显示原图和适应大小，前者没有限制图片尺寸，当图片尺寸大于其父元素的尺寸时。它将显示不全，这时候可以拖动图片查看细节；后者是限制图片尺寸，不超过父元素。
